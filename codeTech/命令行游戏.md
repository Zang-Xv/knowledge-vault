## zork

### 方法一：使用Snap安装

如果您的WSL Ubuntu支持Snap，可以使用以下命令直接安装Zork：

```bash
sudo snap install zork
```

然后使用以下命令运行游戏：

```bash
snap run zork
```

### 方法二：下载游戏文件后用Frotz运行​

如果Snap不可用或您想使用Frotz运行特定的游戏文件，可以：
从IF Archive（网络互动虚构游戏资源库）下载Zork的Z-Machine游戏文件（通常是.z3、.z5等格式）
[Infocom 游戏大全](https://eblong.com/infocom/#zorkmdl)
使用Frotz运行游戏文件：

```bash
frotz let.z3
```

其中let.z3是游戏文件名。确保游戏文件位于您当前的Linux目录中，可以用dir命令查看。​

验证安装
安装完成后，可以用以下命令验证Frotz是否成功安装：

```bash
frotz --version
```

这会显示已安装的Frotz版本信息。​

注意事项
更新软件源：安装前最好先运行sudo apt update确保软件源是最新的​
Snap支持：在某些WSL配置中可能不支持Snap，此时应使用方法二
游戏文件格式：Frotz支持多种Z-Machine游戏文件格式，包括Zork系列的所有版本​
