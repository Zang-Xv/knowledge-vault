# code runner有什么用
Code Runner 相当于一个 轻量级的运行器，特点是：

- 提供一个快捷键（默认 Ctrl+Alt+N）一键运行当前文件，而不用每次右键找菜单。
- 可以运行 选中的部分代码（选中几行 → Ctrl+Alt+N）。
- 输出结果在 OUTPUT 面板 或 终端，而不是只靠调试器。
- 支持多语言（Python、C、C++、Java、JS...），不用针对不同语言装一堆运行扩展。

本质上只是提供了一个在vscode内快速运行代码文件的按钮和快捷键,不需要每次都敲指令.
它不会安装编译器和解释器, 也不配置工具链, 如果系统里没有安装对应的运行环境, 没有运行的命令(如python, javac, g++)可用的话, code runner也无法帮你运行代码文件.
